Background Information

This Application was constructed to provide information to you and others about the impact of covid-19 and displays how covid-19 is affecting your community. We have closely observed and analyzed different data sets and statistics covid-19 cases to display the perfect map for others nationwide to view.


How to properly utilize the tracker

by Lena, Ronesha, Christian and Cesar

The “+” and “-” sign allows those who are viewing to implement a more appealing view for their eyes whether the map needs to be a size smaller or a size bigger.

The text box in the right top corner allows the viewer to type in a particular address or place so that the map can identify the exact county, city and state pertaining to the hospital data for covid-19 that was collected, sources can be more accurate this way.

Covid data can be displayed as well by clicking upon a certain county or state with a mouse.

If you click the double arrow in the top left corner, a menu will appear that allows you to toggle which layers are viewed. Each layer shows different information. The State Borders layer simply allows the user to have more clearly defined state borders so they can easily orient where they are looking at. The Definitive Healthcare Bed Locations shows where there are staffed hospital beds in each county as well as how many total for each county. The COVID active and death rate ratio shows how many total COVID-19 cases divided by how many deaths a county has experienced so the user can see the ratio. The COVID active and confirmed rate ratio shows how many total COVID-19 cases divided by how many cases tested positive a county has experienced so the user can see the ratio.
